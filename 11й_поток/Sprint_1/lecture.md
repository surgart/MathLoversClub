## 1.2 Действительные числа. Числовая прямая

Непустое множество $\mathbb{R}$ называется множеством **действительных чисел**, если:
- на нем определены две операции (**сложение и умножение**)
- и отношение порядка (**не больше**)

На множестве $\mathbb{R}$ выделены два различных элемента $0$ и $1$

$\mathbb{I}.$ **Аксиомы сложения:**

1. **Коммутативность:** $\alpha + \beta = \beta + \alpha$

2. **Ассоциативность:**
   $$(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$$

3. **Свойство нуля:** $0 + \alpha = \alpha$

4. Существование **противоположного числа:**
   $$\forall \alpha \exists \beta := -\alpha := (-1)\alpha : \alpha + \beta = 0$$

По определению **разность:**
$$\alpha - \beta := \alpha + (-\beta)$$

$\mathbb{II}.$ **Аксиомы умножения:**

1. **Коммутативность:** $\alpha \cdot \beta = \beta \cdot \alpha$

2. **Ассоциативность:**
   $(\alpha \cdot \beta) \cdot \gamma = \alpha \cdot (\beta \cdot \gamma)$

3. **Свойство единицы:** $1 \cdot \alpha = \alpha$

4. **Существование обратного числа:**
   $\forall \alpha \neq 0 \, \exists \beta := 1/\alpha \hookrightarrow \alpha \cdot \beta = 1$

$\mathbb{III}.$ Связь сложения с умножением**

1. **Дистрибутивность:**

$$(\alpha + \beta) \cdot \gamma = \alpha \cdot \gamma + \beta \cdot \gamma$$

$\mathbb{IV}.$ **Аксиомы порядка:**

1. Всякое число не больше себя: $a \leq a$

2. Из двух разных чисел одно меньше другого: $a \neq b \Leftrightarrow$ или $a < b$, или $b < a$

3. Если одно число не меньше другого и наоборот, то числа равны:
$a \leq b \wedge b \leq a \leftrightarrow a = b$

4. **Транзитивность порядка:**
$a < b \wedge b < c \Rightarrow a < c$

5. **Связь порядка и сложения:**
Если $a < b$, то $\forall c \Rightarrow a + c < b + c$

6. **Связь порядка и умножения:**
Если $a < b$, то $\forall c > 0 \Rightarrow ac < bc$

7. Свойство **непрерывности:**
пусть $X, Y \subset \mathbb{R}$ — непустые подмножества;
пусть $\forall x \in X$ и $\forall y \in Y$ справедливо $x \leq y$.
Тогда существует такое число $a$, что
$\forall x \in X \, \forall y \in Y \rightarrow x \leq a \leq y$ т.е. число $a$ находится между $X$ и $Y$

### **Числовая прямая**

Между всеми точками $B \in l$ и действительными числами $b \in \mathbb{R}$ существует единственное взаимно-однозначное соответствие (**биекция**):

$$l \ni B \leftrightarrow b \in \mathbb{R},$$

сохраняющее и расстояние ($|OB| = |b|$), и порядок ($D–B–C \Leftrightarrow d < b < c$).

1. отрезок
$[a, b] := \{x \in \mathbb{R} : a \leq x \leq b\}$;

2. интервал
$(a, b) := \{x \in \mathbb{R} : a < x < b\}$;

3. полуинтервалы
$(a, b] := \{x \in \mathbb{R} : a < x \leq b\}$,
$[a, b) := \{x \in \mathbb{R} : a \leq x < b\}$.

Удобно ввести формальные символы

$+\infin, -\infin, \infin$

и по определению считать, что для любого $a ∈ ℝ$ выполняется двусторонняя оценка:

$-\infin < a < +\infin$

Расширенная числовая прямая:

$\overline{ℝ} := \{-\infin\} ∪ ℝ ∪ \{+\infin\}$

На $\overline{ℝ}$ отношение порядка сохраняется!

Мы будем работать

- с **полубесконечными** интервалами
  $(a, +\infty), (-\infty, b)$
- с полубесконечными **полуинтервалами**
  $[a, +\infty), (-\infty, b]$ (**лучами**)
- со всей числовой прямой $\mathbb{R} = (-\infty, +\infty)$

$\Def$ Отрезки, интервалы, полуинтервалы, лучи и всю числовую прямую мы будем называть **промежутками** и обозначать $\langle a, b \rangle$

## 1.3 Расстояния и окрестности

$\Def$ **Расстоянием** между произвольными (точками) $a, b \in \mathbb{R}$ называется неотрицательное число $\rho(a, b) := |a - b|$

$\Def$ $ε$ - **окрестность**:
$$
U_ε(a):=\{x\in\mathbb{R}: \rho(a, x) < ε\} = \\
= \{x \in \mathbb{R}: |a - x| < ε\} = \\
= \{x \in \mathbb{R}: a - ε < x < a + ε\} = \\
= (a - ε, a + ε)
$$

$$
U_ε(-\infin) := \left(-\infin, -\frac{1}{ε}\right) \\
U_ε(+\infin) := \left(\frac{1}{ε}, +\infin\right) \\
U_ε(\infin) := \left(-\infin, -\frac{1}{ε}\right) \cup \left(\frac{1}{ε}, +\infin\right)\\
$$

## 1.4 Точные грани числового множества

Понятие точных граней являются основными для обоснования теории пределов и всего математического анализа.
Они обобщают понятия $\max$ и $\min$ и позволяют доказать, что на $\mathbb{R}$ нет **дырок**

$\Def$ Подмножество $X \subset \mathbb{R}$ называется ограниченным сверху (снизу), если существует такое число $C$, что для любого числа $x \in X$ справедливо неравенство $x \le C (x \ge C)$

$\exists C \in \mathbb{R}: \forall{x} \in X \hookrightarrow x \le C (x \ge C)$

$\Def$ Число $C$ называется **верхней (нижней) гранью** множества $X$

$\Def$ Подмножество, ограниченное и сверху и снизу называется **ограниченным**

$\Def$ Число $M$ называется **точной верхней гранью** множества $X$ ($\sup{X}$), если:
$\forall{M'} < M\; \exists x \in X: x > M'$

$\Def$ **Точная нижняя грань** $m = \inf{X}$:
$\forall{m'} > m\; \exists x \in X: x < m'$

$\Cons$
$\sup{X}$ - это наименьшая из верхних граней
$\inf{X}$ - это наибольшая из нижних граней


$\Th{1}$ Если непустое множество $X$ ограничено сверху (снизу), то существует причем единственный $\sup{X}$ ($\inf{X}$)

$\Disc$

Прежде, чем исследовать математический объект нужно:
**1.** Доказать его **существование**
**2.** Проверить его **единственность** (не является обязательным, но удобнее исследовать единственный объект)

$\Def$ $\max{X}$ - это такое число, что $\max{X} \in X \land \forall{x} \in X \hookrightarrow x \le \max{X}$, если $\sup{X} \in X$, то $\max{X} = \sup{X}$, иначе $\max{X}$ отсутствует.
Для ограниченного сверху множества $\sup{X}$ существует всегда, а $\max{X}$ не всегда.

Аналогично для $\inf$ и $\min$

$\Ex{1}$

Пусть $X = (0, 1]$, тогда $\inf{X} = 0, \max{X} = \sup{X} = 1$, а $\min{X}$ отсутствует

$\Th{2}$ **Теорема о разделении двух множеств**

Пусть $X, Y \subset \mathbb{R}$ - непустые множества.
Причем $\forall x \in X$ и $\forall y \in Y$ справедливо $x \le y$.
Тогда существуют $\sup{X}$ и $\inf{Y}$, причем
$\forall x \in X\;\forall y \in Y \hookrightarrow x \le \sup{X} \le \inf{Y} \le y$

$\Def$ Для неограниченного сверху множества $X$ положим по определению $\sup{X} = +\infin$
Для неограниченного снизу: $\inf{X} = -\infin$

## 2.1 Понятие числовой последовательности

$\Def$ Если каждому натуральному числу $n$ поставлено в соответствие некоторое действительное число $a_n$, то говорят, что определена числовая последовательность $\{a_n\}$.

$\Def$ То есть, числовая последовательность $\{a_n\}$ — это отображение $a: \mathbb{N} \rightarrow \mathbb{R}$.

Числовая последовательность $\{a_n\}$

Элементом последовательности называется пара $(n, a_n)$.

Натуральное число $n$ — номер элемента, $a_n$ — значение $n$-го элемента.

Множество $\{(n, a_n)\}$ всех элементов последовательности счётно и упорядоченно по строгому возрастанию номеров.

Множество $\{a_n\}$ всех значений последовательности может быть как счётным, так и конечным.

$\Ex{2.1}$

1. $a_n = n$, $\{a_n\} = \mathbb{N}$,

2. $a_n = (-1)^n$, $\{a_n\} = \{-1, 1\}$


**Способы задания последовательности:**

**1.** Формулой $a_n = f(n)$

$\Ex{2.2}$ $a_n = n^2$

**2.** Как сумма членов данной последовательности $S_n = a_1 + \ldots + a_n$

$\Ex{2.3}$

1. Если $a_n = a_1 + (n - 1)d$,
тогда $S_n = \frac{2a_1 + d(n - 1)}{2} \cdot n$ — сумма арифметической прогрессии.

2. Если $a_n = a_1 q^{n-1}$,
тогда $S_n = \frac{b_1(1 - q^n)}{1 - q}$ — сумма геометрической прогрессии.

3. Если $a_n = \frac{1}{n}$,
Тогда $S_n = \frac{1}{1} + \frac{1}{2} + \cdots + \frac{1}{n}$ — частичная сумма гармонического ряда.
Формулы в элементарных функциях $S_n$ не существует.

**3.** Рекуррентный способ.
Выражение значения $n$-го элемента последовательности через значения предыдущих элементов.

$\Ex{2.5}$
Пусть $a > 0$ — фиксированное число.
Последовательность задана рекурсивно:
$$
\left\{
\begin{array}{l}
a_0 = a, \\
a_{n+1} = \frac{1}{2} \left(a_n + \frac{a}{a_n}\right)
\end{array}
\right.
$$

$\Def$ Действительное число $a = \lim_{n \to \infty} a_n$ называется **пределом последовательности** $\{a_n\}$, если

**1.** на языке «$ε$-модуль»:
$\forall ε > 0 \ \exists N = N(ε) \in \mathbb{N}: \ \forall n \geq N \ \hookrightarrow \ |a - a_n| < ε$
Т.е. для любого положительного числа $ε$ найдётся такой номер $N$, начиная с которого для всех натуральных $n \geq N$ выполняется оценка $|a - a_n| < ε$.

**2.** на языке «$ε$-окрестности» (годится также для случаев $a=\pm\infin, \infin$):
$\forall ε > 0 \ \exists N = N(ε) \in \mathbb{N}: \ \forall n \geq N \ \hookrightarrow \ a_n \in U_ε(a)$
Т.е. для любого положительного числа $ε$ найдётся такой номер $N$, начиная с которого для всех натуральных $n \geq N$ значения элементов последовательности принадлежат $ε$-окрестности точки $a$.

$\Def:$ Если предел конечен, т.е. $a \in \mathbb{R}$, то последовательность называют **сходящейся**.

$\Disc$
Логическая конструкция Коши
«для любого $ε$ … для всех достаточно больших $n$» формализует интуитивное представление о том, что числа $a_n$ **«стремятся»** к числу а.

**1.** последовательность **не обязательно** имеет предел;
**2.** определение не является **конструктивным**:
   - **НЕ** сказано **КАК** найти предел;
   - сказано, какому характеризующему его свойству предел удовлетворяет.

## 2.2 Задача. Доказательство сходимости последовательности

$\Prob{1}$
Доказать, что $\lim_{n\rightarrow\infin}\frac{1}{n} = 0$

По определению предела:
$\lim_{n \to \infty} a_n = a \Leftrightarrow \forall \varepsilon > 0 \exists N(\varepsilon) \in \mathbb{N}: \forall n \geq N(\varepsilon) \Rightarrow |a - a_n| < \varepsilon$

$\square$
**1)** $|0 - \frac{1}{n}| < ε \Rightarrow \frac{1}{n} < ε \Rightarrow n > \frac{1}{ε}$

**2)** $N(ε) = \lceil\frac{1}{ε} \rceil + 1$

**3)** $\forall{ε > 0}\quad\exists{N(ε)} = \lceil\frac{1}{ε} \rceil+ 1: \forall n \ge N(ε) \hookrightarrow |0 - \frac{1}{n}| < ε \Rightarrow \lim_{n\rightarrow\infin}{\frac{1}{n}}=0$

$\blacksquare$

## 2.3 Свойства предела. Переход к пределу в неравенствах

**Первоначальные свойства:**
**1.** Если предел существует, то он единственный.
**2.** Сходящаяся последовательность ограничена, т.е. ограничено множество ее значений.

**1.** (о переносе неравенства с пределов на значения элементов последовательностей).

Пусть существуют пределы

$\lim a_n = a \in \mathbb{R} \text{ и } \lim b_n = b \in \mathbb{R}$ причем $a < b$

Тогда существует такой номер $N$, начиная с которого (т.е. $n \geq N$) выполняется строгое неравенство $a_n < b_n$

**2.** (о переносе неравенства со значений элементов последовательностей на их пределы).

Пусть существуют пределы

$\lim a_n = a \in \mathbb{R}$ и $\lim b_n = b \in \mathbb{R}$

и существует такой номер $N$, начиная с которого (т.е. $n \geq N$) выполняется строгое (или нестрогое) неравенство $a_n < b_n$ ($a_n \leq b_n$). Тогда имеет место нестрогое неравенство $a \leq b$.

$\Th{2.3}$ (о трёх последовательностях)

Пусть существует такой номер $N$, начиная с которого (т. е. $n \geq N$) выполняется двустороннее неравенство $a_n < b_n < c_n$ (или $a_n \leq b_n \leq c_n$, или другие варианты однотипных неравенств).

Пусть существуют и совпадают пределы $\lim a_n = \lim c_n = a \in \mathbb{R}$.

Тогда $\lim b_n = a$.

## 2.4 Свойства бесконечно малых последовательностей. Арифметические действия со сходящимися последовательностями

$\Def$ Последовательность $a_n$ называется **бесконечно малой**, если её предел $\lim_{n \to \infty} a_n = 0$

**Свойства** бесконечно малых последовательностей:

**1.** Последовательность $a_n$ имеет конечный предел $a \in \mathbb{R}$ только тогда, когда последовательность $b_n := a_n - a$ является бесконечно малой.

**2.** Если последовательности $a_n$ и $b_n$ бесконечно малые, то их сумма и разность $c_n := a_n \pm b_n$ являются бесконечно малыми.

**3.** Если последовательность $a_n$ бесконечно малая, а последовательность $b_n$ ограниченная, то произведение последовательностей $c_n := a_n \cdot b_n$ является бесконечно малой.

**Арифметические операции** сходящихся последовательностей:

**1.** Если $\lim a_n = a \in \mathbb{R}$, $\lim b_n = b \in \mathbb{R}$, то существует $\lim(a_n \pm b_n) = a \pm b$.

**2.** Если $\lim a_n = a \in \mathbb{R}$, то существует $\lim |a_n| = |a|$.

**3.** Если $\lim a_n = a \in \mathbb{R}$, $\lim b_n = b \in \mathbb{R}$, то существует $\lim(a_n \cdot b_n) = a \cdot b$.

**4.** Если $\lim a_n = a \in \mathbb{R}$, $\lim b_n = b \neq 0$, то существует $\lim\left(\frac{a_n}{b_n}\right) = \frac{a}{b}$.

Из условия данной теоремы следует, что для всех достаточно больших номеров $b_n \neq 0$.

## 2.9 Монотонные последовательности

Специфический, весьма узкий класс последовательностей

$\Def$ Последовательность называется **строго (нестрого) возрастающей**, если
$$\forall n \in \mathbb{N} \hookrightarrow a_n < a_{n+1} (\leq)$$

Нестрого возрастающую последовательность называют **неубывающей**.

Аналогично определяют строго (нестрого)
убывающую (невозрастающую)
последовательность.

$\Def$ Последовательность называют **монотонной**, если она или неубывающая, или невозрастающая.

$\Th{2.9}$ Теорема (Карл Вейерштрасс)
Объясняет в чем ценность монотонных последовательностей

Если последовательность неубывающая (невозрастающая), то она имеет предел

$$\lim a_n = \sup\{a_n\} \in \overline{\mathbb{R}}$$

$$\left(\lim a_n = \inf\{a_n\} \in \overline{\mathbb{R}}\right)$$

$\Disc$

Поскольку точные грани всегда существуют, то в случае монотонности имеем гарантированное существование предела.

В этом ценность данного утверждения.

## 3.1 Принцип вложенных отрезков

$\Def$ Последовательность отрезков $\{[a_n, b_n]\}$ называется **вложенной**, если $\forall{n} \in \mathbb{N}: [a_{n+1}, b_{n+1}] \subset [a_n, b_n]$

$\Def$ Последовательность вложенных отрезков $\{[a_n, b_n]\}$ называется **стягивающейся**, если $b_n - a_n \rightarrow 0$ при $n \rightarrow \infin$

Обозначим:
$A := \{a_1, a_2, ...\}$ - множество левых концов всех отрезков
$B := \{b_1, b_2, ...\}$ - множество правых концов всех отрезков

$\Th{3.1}$ **Теорема Кантора**
> **1.** Пересечение последовательности вложенных отрезков не пусто:
> $$
\bigcap_{n=1}^{\infty} [a_n, b_n] = [\sup A, \inf B] \neq \emptyset
$$
> **2.** Пересечение стягивающейся последовательности вложенных отрезков имеет единственную общую точку
> $$
c = \lim_{n\rightarrow\infin}{a_n} = \sup{A} = \lim_{n\rightarrow\infin}{b_n} = \inf{B}
$$

$\square$
**1:**
$$\bigcap_{n=1}^{\infty} [a_n, b_n] = [\sup A, \inf B] \neq \emptyset$$
Поскольку $\forall k, n \in \mathbb{N} \ (k < n) \hookrightarrow$
$$a_1 \leq \dots \leq a_k \leq \dots \leq a_n \leq b_n \leq \dots \leq b_k \leq \dots \leq b_1,$$
то $a_k \leq a_n \leq b_n \leq b_k$
Поэтому $\forall n \in \mathbb{N}$ справедливо
$$a_n \leq \sup A \leq \inf B \leq b_n$$

**2:**
Если последовательность вложенных отрезков стягивающаяся, то имеет место совпадение $c := \sup{A} = \sup{B}$
$\blacksquare$

## 3.2 Частичные пределы. Теорема Больцано-Вейерштрасса

$\Def$ Пусть:
$\{a_n\}$ — произвольная последовательность действительных чисел,
$\{n_k\}$ — произвольная **строго** возрастающая последовательность натуральных чисел, занумерованных индексом $k$.
Тогда последовательность $b_k = a_{n_k} (k \in \mathbb{N})$ называется **подпоследовательностью** последовательности $\{a_n\}$.

$\Ex{}$
1. Сама последовательность является своей подпоследовательностью
2. Последовательность $b_k = 2k - 1$ возрастающих нечетных натуральных чисел является подпоследовательностью последовательности $a_n = n$ натуральных чисел, где $n_k = 2k - 1$

$\Def$ Точка $a \in \overline{\mathbb{R}}$, являющаяся пределом некоторой подпоследовательности, называется **частичным пределом**

$\Ex{}$
Последовательность $a_n = (-1)^n$ имеет два частичных предела $\pm 1$

$\Def$ **Предельным множеством** последовательности называется множество всех частичных пределов этой последовательности и обозначается $L$

$\Def$ Верхним и нижним пределами называют
$$\overline{\lim}{a_n} := \sup L, \quad \underline{\lim} a_n := \inf L$$

$\Th{3.2}$ **Теорема Больцано-Вейерштрасса**
<sup><sup>(теорема о предельной точке, теорема о компактности, лемма о выборе сходящейся последовательности)</sup></sup>
> Ограниченная последовательность $x_n$ имеет по крайней мере хотя бы один частичный предел
<sup>
**Другая формулировка:** Из любой ограниченной последовательности можно выделить сходящуюся подпоследовательность.</sup>

$\square$ <sup>*(методом дихотомии)*</sup>

**1.** В силу ограниченности существует отрезок, целиком содержащий значения последовательности: $[a_0, b_0] \supset \{x_n\}$.
**2.** $[a_0, b_0] = [a_0, (a_0 + b_0)/2] \cup [(a_0 + b_0)/2, b_0]$
выделим подотрезок, содержащий значения бесконечного количества точек последовательности.
**3.** Выделенный подотрезок называем $[a_1, b_1] \subset [a_0, b_0]$ и опять применяем дихотомию: выделяем подотрезок, содержащий значения бесконечного количества точек последовательности.
**4.** И т.д.
Получаем последовательность стягивающихся вложенных отрезков
$[a_0, b_0] \supset [a_1, b_1] \supset \dots \supset [a_k, b_k] \supset \dots ,$ $|b_k - a_k| = \frac{|b_0 - a_0|}{2^k} \xrightarrow[k \to \infty]{} 0 ,$ каждый из которых содержит значения бесконечного количества элементов последовательности.
**5.** Из **теоремы Кантора** получаем точку $x_0$, каждая окрестность которой содержит значения бесконечного количества элементов последовательности.
**6.** Следовательно можно определить подпоследовательность $x_{n_k}$, сходящуюся к числу $x_0$.

$\blacksquare$

## 3.3 Критерий Коши

$\Def$ Последовательность $\{a_n\}$ называется фундаментальной, если:
$$
\forall{ε} > 0, \exists N \in \mathbb{N}: \forall{n} \ge N \; \forall{p} \in \mathbb{N} \hookrightarrow |a_n - a_{n+p}| < ε
$$
$\NB$ Понятие фундаментальности оценивает **близость значений элементов друг к другу**, а не к пределу

$\Th{3.3}$ **Критерий Коши**
> Последовательность является сходящейся тогда и только тогда, когда она фундаментальна

Важность критерия Коши в том, что его формулировка не использует сам предел

$\square$
$\Rightarrow:$ Из неравенства треугольника следует:
если $a = \lim{a_n}$, то
$$
\forall{ε} > 0, \exists N \in \mathbb{N}: \forall{n} \ge N \; \forall{p} \in \mathbb{N} \hookrightarrow \\
\hookrightarrow |a_n - a_{n+p}| \le |a_n - a| + |a - a_{n+p}| \le \frac{ε}{2} + \frac{ε}{2} = ε
$$

$\Leftarrow:$
**1.** Из фундаментальности последовательности следует ее **ограниченность**.
Для $ε = 1 \; \exists N \in \mathbb{N}: \forall{n} > N \hookrightarrow a_N - 1 < a_n < a_n + 1$
Значит, множество всех значений последовательности, начиная с номера $N$ ограничено.
Поскольку множество $\{a_1, ..., a_{N-1}\}$ конечно, то вся последовательность ограничена.

**2.** Из ограниченности последовательности следует **существование конечного частичного предела** $a$ (теорема Больцано-Вейерштрасса)

**3.** Частичный предел $a$ - **единственный**
Если допустить противное, получим противоречие с фундаментальной последовательностью.

**4.** Число $a$ - предел последовательности
Если допустить противное, получим существование ещё одного частичного предела, отличного от $a$

$\blacksquare$

$\NB$
